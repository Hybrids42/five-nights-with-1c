# Проект комманды «6055ы» для хакатона «1С Бот»

## Кейс

Игровой телеграм-бот в формате текстового квеста на 1С

## Описание

Представьте, что вы — ночной охранник в офисе 
1С. Внезапно система сходит с ума, и чтобы 
выжить, вам нужно за одну ночь понести на себе тяжкую ношу IT-специалиста: стать тестировщиком/разработчиком. 
Мы создали атмосферный портал для 
погружения в мир IT, где обучение чувствуется на кончиках пальцев.

## Плюсы проекта

- Низкий порог входа — нужен только Telegram
- Эффективная геймификация — рутинные задачи превращаются в игру на выживание
- "Примерочная" профессий в экстремальных условиях
- Интересное повествование
- Проект работает :)

## Запуск

Запустить 1С, в нём выбрать режим «Игра», вставить токен бота и нажать «Запустить бота»
